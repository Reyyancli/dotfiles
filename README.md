# Dotfiles
Config Files for stuff!

yeah.. thats it.

## Whats included?
- Love from me ⊂(￣▽￣)⊃
- [i3](https://github.com/i3/i3) Config
- [picom](https://github.com/yshui/picom) Config
- [Hyprland](https://github.com/hyprwm/Hyprland) Config
    - Currently configured with hyprscroller which is now archived and broken so you may have to fix the bindings yourself.
- [scroll](https://github.com/dawsers/scroll) Config
- [Waybar](https://github.com/Alexays/Waybar) Config
    - This config is taken and modified from [this git repo](https://github.com/zihadmahiuddin/yolkfiles/tree/main/eggs/waybar)
- [polybar](https://github.com/polybar/polybar) Config
- [niri](https://github.com/YaLTeR/niri) Config
- [neovim](https://github.com/neovim/neovim) Config
    - Plugins not included
- [fuzzel](https://github.com/catppuccin/fuzzel) Config
- [starship](https://github.com/starship/starship) Config
    - It is recommended that you install a nerdfont such as Hack 
- [zsh](https://github.com/zsh-users/zsh) Config 
    - Plugins also not included
